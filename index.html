Sobre nosotros


Al usar nuestro hub tutorial, puedes aprender y ejectuar código de computadora de forma experimental para ver que funciona y que no antes de implementarlo. Diagramas paso a paso son probablemente los más populares y fáciles de seguir para mostrar como doblar cosas con papel. Para ayudarte en tu búsqueda, hemos juntado la más grande colección de diferente lenguaje técnico en internet.

HTML

  1. HTML es el lenguaje estándar para crear

    páginas web.

  2. HTML describe la estructura de una página

    web.

  3. HTML consiste en una serie de elementos.

CSS

  • CSS es el lenguaje que usamos para dar

    estilo a un documento HTML.

  • CSS describe como los elementos HTML debe

    ser mostrados.

  • CSS ahorra mucho trabajo. Puede controlar

    el diseño de

    múltiples páginas web, todas a la vez.

Javascript

  1. JavaScript es el lenguaje de programación

    de la web.

  2. JavaScript es fácil de aprender.

PHP

  • PHP 7 es la versión más estable y reciente.

  • Es una herramienta poderosa para crear páginas

    web dinámicas e interactivas

  • PHP PHP es una alternativa apmpliamente usada,

    gratis y eficiente para competidores como

    Microsoft ASP.

Python

  1. Python puede ser usado en un servidor para

    crear aplicaciones web.

  2. Python puede ser usado junto con algún

    software para crear workflows.

  3. Python tiene una sintaxis simple similar

    al idioma inglés.

React Native

  • React Native soporta plataformas iOS

    y Android.

  • Con React Native, no creas una aplicación

    web, HTML5 o hibrida.

Bootstrap

  1. Bootstrap es el marco de trabajo más popular

    de HTML, CSS y JavaScript para desarrollar

    sitios web de fácil respuesta y móviles.

  2. ¡Bootstrap es completamente gratis para

    descargar y usar!

p5.Js

  1. p5.js es una biblioteca JavaScript para

    programación creativa e incluso para artistas,

    diseñadores, principiantes y ¡quien sea!

  2. p5.js es gratis y abierto porque creemos

    que el software, y las herramientas para

    aprender, deben ser accesibles para todos.
